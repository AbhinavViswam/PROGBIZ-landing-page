# Project README

## Overview

This project consists of both a **frontend** and **backend**,

```bash
npm run dev
```

Running this command for both frontend and backend.

---

## Features

* Full‑stack application with separately running frontend and backend
* Admin login system
* Edit mode available only to authenticated admins
* Navigation bar shows an **Edit** option once logged in as admin
* Edit page allows modification of project content

---

## Installation & Setup

1. **Clone the repository:**

```bash
git clone https://github.com/AbhinavViswam/PROGBIZ-landing-page.git
```

2. **Install dependencies seperately for both frontend & backend:**

```bash
npm install
```

3. **Start the project:**

```bash
npm run dev
```

You have to do this in both backend folder as well as frontend folder.

---

## Admin Login

To access admin‑only functionality (including the **Edit** page), log in using:

* **Email:** `admin@mail.com`
* **Password:** `admin123`

Once logged in, an **Edit** button will appear near the navigation bar, taking you to the edit interface.
